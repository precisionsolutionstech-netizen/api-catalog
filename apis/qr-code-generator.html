QR Code Generator API

QR code generator, creator, and barcode API. Generate configurable QR codes with logos, colors, error correction. PNG, SVG, or WEBP output. Scan verification, quality score. Deterministic, production-ready.

Why use this API?

Generate QR codes for tickets, payments, menus, or marketing without hosting your own generator.

What the API does

POST /generate with payload (e.g. content, size). Returns image or URL to QR image.

About this API

QR Code Generator API

The QR Code Generator API creates highly configurable, production-ready QR codes with built-in scan reliability analysis. It supports advanced customization such as logos, colors, error correction levels, and output formats, while validating the final QR code for scannability and quality.

What This API Does

  • **Generates QR codes** from text or URLs with configurable size and error correction levels
  • **Embeds logos** in the center (PNG/WEBP only) with automatic scaling and padding
  • **Customizes appearance** with custom foreground and background colors
  • **Validates quality** with scan reliability verification and contrast analysis
  • **Supports multiple formats** — PNG (default), SVG, and WEBP
  • **Provides warnings** for non-fatal issues that may affect scan reliability

The API is deterministic and rule-based. Same input always produces identical output. QR codes are validated for scannability.

Key Features

  • **Stateless & Deterministic** — No data stored; 5MB max payload
  • **Error correction** — L, M, Q, H (max data length varies)
  • **Logo support** — Base64 PNG/JPG/GIF/WEBP; scale 0.1–0.3
  • **Size** — 64–4096 px; quiet zone 0–10 modules
  • **Quality** — `scanVerified`, `qualityScore`, `warnings`

Use Cases

  • **URL encoding** — Links, deep links, UTM tracking
  • **Branded QR codes** — Custom colors and logos
  • **Production use** — Reliable, validated output
  • **Multiple formats** — PNG for web, SVG for print, WEBP for modern stacks
